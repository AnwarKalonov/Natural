# Natural

Natural is a collaborative development environment where users can build software using plain English, traditional code, or a combination of both.

It is designed to make software development more accessible while still giving users full control over structure, logic, and implementation.

---

## Overview

Natural allows users to describe functionality in plain language and have it translated into working code. At the same time, it provides a full code editor, live preview, and collaboration tools so projects remain structured and maintainable.

This is not a chat interface layered on top of AI. Natural is built as a workspace that manages files, tracks changes, and supports real development workflows.

---

## Core Features

### Natural Language Input

* Write instructions in plain English
* Generate code based on intent
* Refine and iterate using follow-up instructions

### Code Editor

* Multi-file project structure
* Syntax highlighting and formatting
* Manual editing alongside AI-assisted changes
* Diff previews before applying AI modifications

### AI-Assisted Development

* Refactor, optimize, and fix code
* Apply changes to selected code, files, or entire projects
* Maintain consistency across the codebase
* Avoid full file rewrites unless explicitly requested

### Live Preview and Visual Editing

* Real-time application preview
* Select elements directly from the UI
* Modify styles, layout, and behavior visually
* Sync visual edits back to code

### Collaboration

* Shared projects for teams
* Real-time editing
* Team chat and communication
* Role-based access control

### Version History

* Track all changes over time
* Separate AI-generated changes from manual edits
* Revert to previous versions when needed

---

## How It Works

1. Create a project
2. Write code or describe functionality in plain English
3. Generate or modify code using AI
4. Preview changes in real time
5. Collaborate and iterate

Natural keeps the project structured so changes remain predictable and manageable.

## Tech Stack

* Frontend: React with Tailwind CSS
* Backend: Node.js or serverless functions
* Database: PostgreSQL
* AI: Google AI Studio
* Hosting: Vercel or a cloud provider

## Vision

Natural is built to shift development from writing syntax to expressing intent.

The goal is to make software development:

* Faster to iterate
* More collaborative
